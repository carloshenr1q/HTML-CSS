Aqui tempos exemplo simples de código em javascript 

                                            /* Estilo geral */
                        body {
                            font-family: Arial, sans-serif;
                            background-color: #f4f4f4;
                            margin: 0;
                            padding: 20px;
                            color: #333;
                        }

                        /* Estilizando um título */
                        h1 {
                            color: #333;
                            text-align: center;
                        }

                        /* Estilizando parágrafos */
                        p {
                            line-height: 1.6;
                            margin: 10px 0;
                        }

                        /* Estilizando um botão */
                        button {
                            background-color: #008cba;
                            color: white;
                            border: none;
                            padding: 10px 20px;
                            font-size: 16px;
                            cursor: pointer;
                            border-radius: 5px;
                            transition: background-color 0.3s ease;
                        }

                        button:hover {
                            background-color: #005f5f;
                        }
